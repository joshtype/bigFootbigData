Javascript Charts

Below are three charts created using Javascript + Fetch API + Chart.js. Visit the Walkthrough page for more detailed info on how data was prepared.

Pie Chart: Proportion of Sightings per Class A, B, and C

The above chart is a pie chart depicting classifcation grade data points as pie wedges.
From this, it can be observed that almost every sighting is graded as Class A or Class B. An extremely slim portion of sightings are graded as Class C. One might infer from this that evidence of class C sightings aren't easily distinguished from evidence produced by non-bigfoot fauna.
