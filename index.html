What is bF(bD)?

This project focuses on how Javascript and Python can be utilized for dynamic client-side data analysis, visualization, and machine learning prediction. Using a tongue-in-cheek dataset (source: Kaggle.com), the purpose of this site is to demonstrate that Javascript is not merely a valid option for data science. Combining Python's ability to read, clean, and write data, with Javascript's DOM manipulation and ease of publishing to the web, has the potential to transform data science from its normally unseen back-end role into a transparent and interactive engagement between client and server.

Please visit the Walkthrough page for detailed information regarding the libraries and methods I employed to accomplish my goals. Included are screenshots, code excerpts, and files for download.

Please visit the Visualizations page to view a variety of charts that were created using the Javascript library Chart.js from data read and cleaned in Python. They will highlight how visually representating data appropriately can help users interpret data and develop inferences that may not have been possible from raw textual data.

Please visit the Predictions page to view machine learning predictions for certain data created using Tensorflow's complex neural networks. Such tools are now accessible for client-side programming via CDN and Tensorflow's Javascript library, Tensorflow.js.
